# MOOD — 美学知识库

> 激发美的灵感 · 从灵感到认知

MOOD 是一个图文社媒平台，融合灵感流与美学知识图谱，帮助用户不只是收藏图片，而是真正理解「为什么好看」。

## 功能

- 📸 **灵感层** — 图文/视频内容流，点赞、评论、收藏、转发
- 📚 **知识层** — 每张图背后的色彩理论、材质语言、文化参考
- 🗂 **Learning Library** — 色彩、时尚、室内、摄影、电影美学系统课程
- 👤 **个人主页** — 创作内容 + 收藏夹分类管理

## 技术栈

- [Next.js 14](https://nextjs.org/)
- [React 18](https://react.dev/)
- [Vercel](https://vercel.com/) 部署

## 本地运行

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)

## 部署

连接 GitHub 仓库到 Vercel，自动部署。
